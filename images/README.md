# Image assets

This directory and sub-directory contains photographs, documentation images and certification assets used in the Thin-Pod rev 0.1 release documentation.

The OSHWA certification mark for the certified release is held at:

```text
certification/oshwa-uk000091.png
```

The mark identifies **Thin-Pod rev 0.1**, certified under OSHWA UID **UK000091** on **28 May 2026**. It is displayed in the repository `README.md` and links to the public OSHWA certification record.
